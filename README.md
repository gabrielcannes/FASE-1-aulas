## 🚀 Como Usar

### Pré-requisitos

- Python 3.x
- Jupyter Notebook/JupyterLab
- Ambiente virtual (recomendado)

### Configuração do Ambiente

1. **Clone ou baixe** este repositório
2. **Crie um ambiente virtual** (opcional mas recomendado):
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Linux/Mac
   # ou
   .venv\Scripts\activate     # Windows
   ```

3. **Instale as dependências** (se houver requirements.txt futuramente):
   ```bash
   pip install -r requirements.txt
   ```

### Executando os Notebooks

1. **Navegue até o diretório da aula desejada**:
   ```bash
   cd "Aulas-Fase 1/aula5"
   ```

2. **Execute o Jupyter**:
   ```bash
   jupyter notebook
   # ou
   jupyter lab
   ```

3. **Abra o notebook** correspondente à aula (ex: `Aula5-Fase1.ipynb`)
